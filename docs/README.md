## 概述

Borax 是一个的 Python3 开发工具集合库,涉及到：

 - 设计模式示例
 - 数据结构

## 安装

Borax 要求 Python 的版本至少为 3.5 以上。

使用 *pip* ：

```shell
$ pip install borax

```

或者使用开发代码

```shell
git clone https://github.com/kinegratii/borax.git
cd borax
python setup.py install
```
